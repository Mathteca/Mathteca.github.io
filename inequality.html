Jensen's Inequality

If $f(x)$ is a convex function on the interval $I$ and for every $x_1,x_2,\dots,x_n\in I$, we have $$f(\frac{x_1+x_2+\dots +x_n}{n})\leq \frac{f(x_1)+f(x_2)+\dots +f(x_n)}{n}$$ If $f(x)$ is a concave function on the interval $I$ and for every $x_1,x_2,\dots,x_n\in I$, we have $$f(\frac{x_1+x_2+\dots +x_n}{n})\geq \frac{f(x_1)+f(x_2)+\dots +f(x_n)}{n}$$ The equality holds if and only if $x_1=x_2=\dots=x_n$.
