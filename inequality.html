AM-GM Inequality

For nonnegative real number $x_1,x_2,\dots,x_n$, $$max\{x_1,x_2,\dots,x_n\} \geq \sqrt{\frac{x_1^2 + x_2^2 + \cdots + x_n^2}{n}} \geq \frac{x_1 + x_2 + \cdots + x_n}{n} \geq \sqrt[n]{x_1 x_2 \cdots x_n} \geq \frac{n}{\frac{1}{x_1} + \frac{1}{x_2} + \cdots + \frac{1}{x_n}} \geq min\{x_1,x_2,\dots,x_n\}$$ The equality holds if and only if $x_1 = x_2 = \cdots = x_n$.
Remark: The weighted mean version of this inequality holds.

AM-GM Inequality

for any real numbers $a_1, a_2, \ldots, a_n$ and $b_1, b_2, \ldots, b_n$, $$(a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \geq (a_1b_1 + a_2b_2 + \cdots + a_nb_n)^2$$
(APMO 1991 #3) Let $a_1$, $a_2$, $\cdots$, $a_n$, $b_1$, $b_2$, $\cdots$, $b_n$ be positive real numbers such that $a_1 + a_2 + \cdots + a_n = b_1 + b_2 + \cdots + b_n$. Show that $$\frac {a_1^2}{a_1 + b_1} + \frac {a_2^2}{a_2 + b_2} + \cdots + \frac {a_n^2}{a_n + b_n} \geq \frac {a_1 + a_2 + \cdots + a_n}{2}$$

Jensen's Inequality

If $f(x)$ is a convex function on the interval $I$ and for every $x_1,x_2,\dots,x_n\in I$, we have $$f(\frac{x_1+x_2+\dots +x_n}{n})\leq \frac{f(x_1)+f(x_2)+\dots +f(x_n)}{n}$$ If $f(x)$ is a concave function on the interval $I$ and for every $x_1,x_2,\dots,x_n\in I$, we have $$f(\frac{x_1+x_2+\dots +x_n}{n})\geq \frac{f(x_1)+f(x_2)+\dots +f(x_n)}{n}$$ The equality holds if and only if $x_1=x_2=\dots=x_n$.
